# Installation and Runtime

Definers includes system-level helpers for install policy, process execution, paths, downloads, and runtime control.

## Start Here

- `definers.system`
- `definers.system.installation`
- `definers.system.paths`
- `definers.system.processes`

## Best Fit

- optional dependency installation flows
- path validation and secure path handling
- controlled subprocess execution
- environment-sensitive runtime setup

Use this layer when the hard part of the project is reliability around the application, not the model itself.
