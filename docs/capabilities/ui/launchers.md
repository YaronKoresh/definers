# Launchers

Definers exposes focused launcher surfaces instead of forcing every workflow through one overloaded interface.

## Core Entry Point

- `definers.ui.gui_entrypoints.start(project)`
- CLI surface: `definers start <project>`

## Representative Launchers

- `chat`
- `audio-mastering`
- `audio-stems`
- `audio-analysis`
- `image-generate`
- `video-composer`
- `video-lyrics`
- `train`

Use launchers when the goal is a user-facing workflow surface rather than a low-level library call.
