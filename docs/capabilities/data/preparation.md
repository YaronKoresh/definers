# Data Preparation

Definers provides one preparation layer for loading, shaping, tokenizing, vectorizing, and batching heterogeneous inputs.

## Start Here

- `definers.data.preparation`
- `definers.data.vectorizers`
- `definers.data.tokenization`

## Best Fit

- mixed feature and label loading
- text tokenization and vectorization
- tensor dataset construction
- shape normalization before training or inference

This is the right entry point when the main problem is turning raw inputs into consistent model-ready structures.
