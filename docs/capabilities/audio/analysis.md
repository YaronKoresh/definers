# Audio Analysis

Definers exposes audio analysis workflows for tempo, structure, spectral behavior, and descriptive summaries.

## Start Here

- Python modules under `definers.audio.analysis`
- Focused launcher surface: `audio-analysis`

## Typical Outputs

- BPM and beat information
- spectrum and RMS summaries
- feature payloads for downstream ML or UI flows
- human-readable analysis summaries

Use this path when you need diagnostics, not transformation.
